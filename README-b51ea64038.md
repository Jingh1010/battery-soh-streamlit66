# 🔋 蓄电池健康状态（SOH）预测系统

基于机器学习的电池健康状态预测完整解决方案，支持多种算法、数据可视化和实时预测。

## 📋 项目概述

本项目实现了一个完整的机器学习工程流程，用于预测蓄电池的健康状态（State of Health, SOH），包含数据预处理、特征工程、模型训练、评估和可视化等完整环节。

### 核心功能

- ✅ **多模型支持**：随机森林、梯度提升、支持向量机、线性回归
- 📊 **完整数据流程**：数据加载 → 预处理 → 特征工程 → 模型训练 → 评估
- 🎨 **丰富可视化**：数据分布、相关性分析、预测结果对比、特征重要性
- 🔮 **实时预测**：输入电池参数，即时获取SOH预测结果
- 📈 **模型对比**：多模型性能对比，自动选择最优模型

### 评估指标

- **MSE（均方误差）**：衡量预测偏差
- **MAE（平均绝对误差）**：平均绝对偏差
- **R²（决定系数）**：模型解释能力

## 📂 数据格式要求

支持上传CSV格式的电池数据集，必须包含以下列：

| 列名 | 说明 | 单位 | 范围 |
|------|------|------|------|
| Cycle | 充放电循环次数 | 次 | 1-5000 |
| Voltage | 电池电压 | V | 2.5-4.5 |
| Current | 电池电流 | A | 0.1-10.0 |
| Temperature | 电池温度 | ℃ | -20~80 |
| Capacity | 电池容量 | Ah | 0.5-5.0 |
| SOH | 健康状态（目标变量） | % | 0-100 |

**示例数据格式：**

```csv
Cycle,Voltage,Current,Temperature,Capacity,SOH
1,3.98,1.05,25.3,3.00,100.0
50,3.95,1.02,26.1,2.95,98.5
100,3.92,0.98,27.3,2.88,96.0
...
```

## 🎯 特征工程

系统自动从原始数据中提取以下高级特征：

1. **容量衰减率**：相对于最大容量的衰减百分比
2. **电压变化率**：电压相对于前一时刻的变化
3. **循环效率**：基于循环次数的效率指标
4. **温度归一化**：温度的标准化值（Z-score）

## 🤖 支持的机器学习模型

### 1. 随机森林 (Random Forest)
- **优势**：集成学习方法，对非线性关系建模能力强，不易过拟合
- **适用场景**：数据特征复杂，存在非线性关系

### 2. 梯度提升 (Gradient Boosting)
- **优势**：逐步提升弱分类器，预测精度高
- **适用场景**：追求高预测精度，数据量适中

### 3. 支持向量机 (SVR)
- **优势**：适合小样本数据，泛化能力强
- **适用场景**：数据量较小，注重泛化性能

### 4. 线性回归 (Linear Regression)
- **优势**：简单高效，可解释性强
- **适用场景**：特征与目标变量呈线性关系

## 💡 使用建议

### 数据准备
- 确保数据格式正确，包含所有必需列
- 建议数据量 ≥ 500条以获得稳定结果
- 注意处理缺失值和异常值

### 模型选择
- 数据量小（<1000）：优先使用SVR
- 数据量中等（1000-5000）：随机森林或梯度提升
- 追求可解释性：线性回归
- 追求最高精度：梯度提升

### 参数调整
- 测试集比例：0.15-0.20为宜
- 验证集比例：0.10-0.15为宜
- 根据验证集性能调整超参数
